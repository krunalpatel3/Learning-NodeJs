### Methods Used 

1) Post /AddNote api 

parameter:- 
{
    "title": "My note 1",
    "description":"note description 1"
}

2) Get /GetNotes
3) Delete /DeleteNote/1 here send index of notes Array list 
4) Patch /UpdateNote/1 here send index of notes Array list 

parameter:- 
{
    "title": "My note Update",
    "description":"note description Update"
}
